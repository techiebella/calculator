# 🚀 ByteForge Calculator

A modern, responsive, and feature-rich calculator built using **HTML, CSS, and JavaScript**. Designed with a beautiful glassmorphism interface, dark/light themes, scientific functions, memory operations, and calculation history.

## ✨ Features

### 🧮 Basic Calculator
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- Decimal Support

### 🔬 Scientific Calculator
- sin(x)
- cos(x)
- tan(x)
- Square Root (√)
- Power (x²)
- Exponent (xʸ)
- Factorial (!)
- Pi (π)

### 🎨 Modern UI
- Dark Mode
- Light Mode
- Glassmorphism Design
- Smooth Animations
- Neon Effects
- Mobile Responsive

### 📜 History Panel
- Stores Previous Calculations
- Local Storage Support
- Clear History Option

### ⌨️ Keyboard Support
- Enter = Calculate
- Backspace = Delete
- Escape = Clear
- Supports Numeric Keys

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/techiebella/calculator.git
```

### Open Project

```bash
cd calculator
```

Then simply open:

```text
index.html
```

in your browser.
